About

This is a website that allows you to see info about scammers in the game Rucoy Online. The data will be used to create a list of scammers that will be available to everyone. The goal is to make it easier for players to find and avoid scammers.

The code is open source and available on GitHub. If you want to check it out or contribute, you can find it here!
